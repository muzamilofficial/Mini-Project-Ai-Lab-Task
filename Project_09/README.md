# Car Price Prediction - Exploratory Data Analysis (EDA)

## Overview
This project performs **Exploratory Data Analysis (EDA)** on a car dataset to understand key patterns and variables affecting **car prices**.  
The analysis prepares the data for future **machine learning models** that aim to predict car prices.

---

## Dataset
- **File:** `car_price_prediction.csv`  
- **Rows:** 19,237  
- **Columns:** 18  

### Main Features
- **Price** (Target variable)  
- **Manufacturer** and **Model**  
- **Production Year**  
- **Category** and **Fuel Type**  
- **Engine Volume**, **Cylinders**, **Mileage**  
- **Gear Box Type**, **Drive Wheels**, **Airbags**

---

## Analysis Performed
1. **Data Loading** – Imported the dataset using `pandas`.  
2. **Data Overview** – Checked data shape, column types, and previewed entries.  
3. **Missing Values** – Verified and confirmed no missing data.  
4. **Statistical Summary** – Generated descriptive statistics to understand variable distributions.
